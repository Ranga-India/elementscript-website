Features

Why ElementScript?

Simple Types

Only `string` and `number`. No complex data structures to worry about.

Strict Constraints

Types define their limits, like `string_10` (10 chars) or `number_1` (1 digit).

Minimal Control Flow

Just `if` (no `else`) and a simple `repeat` loop. Forces clean, linear logic.

Clear Syntax

Uses intuitive keywords like `display`, `-and-`, and `-or-`.

See It in Action

ElementScript code is designed to be readable and straightforward.

hello.els 

# --- "Hello, World!" in ElementScript ---

string_100 message = "Hello, World!"
number_1 counter = 1

if (counter > 0) -and- (counter < 10) {
    display message
}


repeat i from 1 to 3 {
    display "Loop: " + i
}
